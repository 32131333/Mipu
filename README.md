Mipu is the one-page web-application and also a social network. His philosophy is like to Twitter and TikTok

## More information
1. Vertical ports is named as **Sparks** or **mipuadv_posts**(codename)

## Run
1. Just write `npm i`
2. Run `node index`
3. And wait the `bundle.js` file creation!! :D
4. Go to `/languages/ru.json` and update the API's url to Mipu's domains
